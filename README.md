# AsioMarketing Astro

Astro starter for the AsioMarketing site with a minimal page and layout scaffold.

## Scripts
- `npm install` тАУ install dependencies
- `npm run dev` тАУ start the dev server
- `npm run build` тАУ create the production build
- `npm run preview` тАУ preview the built site

## Structure
- `src/pages` тАУ route files (currently `index.astro`)
- `src/layouts` тАУ shared layout wrapper
- `src/styles` тАУ global styles
- `public` тАУ static assets such as favicons
